# Blind listening review template

Reviewer must see only the selected WAV and this template before freezing the review.

```text
reviewer_id:
review_date:
audio_sha256:
full_listens:
partial_replays:
source_text_seen: false
asr_output_seen: false

TRANSCRIPT:

UNCERTAINTY MARKS:

NON-SPEECH / PAUSES:

review_frozen_at:
review_sha256:
```

If source text or ASR output was seen before freeze, record the actual condition and do not call the review blind.
